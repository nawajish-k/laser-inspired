## Laser inspired

A cinematic frontend animation project inspired by Laser by Sony Music, featuring custom reveal effects, smooth transitions, and a responsive landing page experience.


### ✨ Overview

This project explores modern web animation techniques by creating an immersive landing page with cinematic visuals and engaging reveal animations.

The focus of this project was to experiment with:
- Smooth page reveal animations
- Modern UI composition
- Responsive layouts


### 🎨 Inspiration

Inspired by the visual style and animation approach of **Laser by Sony Music**.


### 🚀 Features

- Cinematic landing page experience
- Smooth reveal animations
- Responsive design for different screen sizes


### 🛠️ Built With

- HTML
- CSS
- JavaScript
- GSAP
